learn katakana

This site is supposed to help you learn Katakana by reading real words.

The content is based on the titles of articles in the Japanese Wikipedia that are only in English.

Because it's automatically generated, right now there are some rather weird words in here (for instnace, Japanese transliterations of the names of ancient Egyptian gods, small towns in Botswana, etc etc). I'm going to have to figure out some way to weed out the words that would probably be meaningless to the learner even if they were written in the Roman script in the first place.

Still, the levels are "programmatic" in the sense that each level contains only the letters that have been seen before, plus one additional letter. So for someone trying to improve their command of Katakana it might be a fun thing to try to see how many of the words are sensible.

Needless to say this is a work in progress, but if you want to click around, try starting here.
